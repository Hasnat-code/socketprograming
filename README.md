# SocketPrograming

A simple Java repository that show how socket programing work in computer networking

## Overview

This repository contains Java source code for the "socket programing" project. It is a small example project intended to demonstrate basic client and server talk.

## Requirements

- Java 8 or higher
- Maven or Gradle (if a build tool is used)

## Build & Run (Maven)

If the project uses Maven:

1. Build:

   mvn clean package

2. Run (replace MainClass with your main class fully qualified name):

   java -cp target/<artifact>-<version>.jar com.example.MainClass


## Build & Run (Gradle)

If the project uses Gradle:

1. Build:

   ./gradlew build

2. Run (replace MainClass with your main class fully qualified name):

   java -cp build/libs/<artifact>.jar com.example.MainClass
